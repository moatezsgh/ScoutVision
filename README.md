# ScoutVision

Talend ETL project for data extraction, transformation, and loading into a structured data warehouse.

## Tools Used
- Talend Open Studio
- PostgreSQL
- SQL
- Power BI

## Project Goal
This project aims to build ETL workflows to prepare clean and structured data for reporting, analytics, and decision-making.

## Current Progress
- Data integration
- ETL job development
- Data transformation
- Data loading into database
